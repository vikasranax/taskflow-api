# TaskFlow API

A simple REST API for managing tasks, built with FastAPI and containerized with Docker.

## What it does

Basic CRUD operations for tasks — create, read, update, delete. Built this to practice working with FastAPI, Docker, and setting up a proper backend project structure.

## Tech used

- Python / FastAPI
- Uvicorn (server)
- Docker + Docker Compose
- Pydantic for data validation

## Project structure

```
taskflow/
├── app/
│   ├── main.py
│   ├── routes.py
│   └── models.py
├── Dockerfile
├── docker-compose.yml
└── requirements.txt
```

## How to run it

You need Docker installed. Then:

```bash
git clone https://github.com/vikasranax/taskflow-api.git
cd taskflow-api
docker-compose up --build
```

API runs on `http://localhost:8000`

Docs (auto-generated by FastAPI) are at `http://localhost:8000/docs`

## API endpoints

| Method | Endpoint | What it does |
|--------|----------|---------------|
| GET | /api/tasks | List all tasks |
| POST | /api/tasks | Create a task |
| PUT | /api/tasks/{id} | Update a task |
| DELETE | /api/tasks/{id} | Delete a task |

## Example

